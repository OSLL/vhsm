STORAGE TESTS

INIT DATABASE

Проверяется работа функции initDatabase. Тесты:
	- создание БД в пустом файле - успех;
	- создание БД по несуществующему пути - неудача;
	- открытие БД внешними средствами - успех;
	- проверка наличия таблиц в БД - успех;
	
CREATE USER

Проверяется работа функции createUser. Тесты:
	- создание нового пользователя - успех;
	- создание пользователя с занятым именем - неудача;
	- создание еще одного пользователя - успех;
	- открытие БД внешними средствами - успех;
	- проверка количества пользователей в БД - успех;

LOGIN

Проверяется работа функций loginUser/logoutUser. Тесты:
	- регистрация пользователя в хранилище - успех;
	- повторная регистрация - успех;
	- регистрация несуществующего пользователя - неудача;
	- регистрация пользователя с неверным паролем - неудача;
	- регистрация следующего пользователя - успех;
	
IMPORT KEY

Проверяется работа функции importKey, включающей генерацию ключей/id. Тесты:
	- импорт ключа с заданным id - успех;
	- импорт ключа с занятым id - неудача;
	- импорт ключа незарегистрированным пользователем - неудача;
	- генерация ключа и id - успех;
	- проверка количества ключей - успех;
	
DELETE KEY

Проверяется работа функции deleteUser. Тесты:
	- удаление ключа с заданным id - успех;
	- удаление ключа с несуществующим id - неудача;
	- проверка количества ключей в БД после удаления ключа - успех;
	- удаление ключа незарегистрированным пользователем - неудача;
	
PRIVATE KEY

Проверяется работа функции getUserPrivateKey. Тесты:
	- импорт ключа и генерация id - успех;
	- извлечение ключа с заданным id - успех;
	- проверка извлеченного ключа - успех;
	- извлечение ключа незарегистрированным пользователем - неудача;
	- извлечение ключа с несуществующим id - неудача;
	
KEY COUNT

Проверяется работа функции getKeyIdsCount. Тесты:
	- проверка количества ключей в БД - успех;
	- вызов функции незарегистрированным пользователем - неудача;
	
KEY IDS

Проверяется работа функции getKeyIds. Тесты:
	- проверка id, возвращенных функцией - успех;
	- вызов функции незарегистрированным пользователем - неудача;
	
KEY INFO

Проверяется работа функции getKeyInfo. Тесты:
	- проверка информации о ключах, возвращенной функцией - успех;
	- вызов функции незарегистрированным пользователем - неудача;
