VHSM TESTS

LOGIN

Проверяется способность VHSM поддерживать список пользователей в акутальном сстоянии. Тесты:
	- первичная регистрация пользователя - успех;
	- двойная регистрация пользователя - успех;
	- завершение сеанса - успех;
	- двойное завершение сеанса - неудача;
	- закрытие сессии - успех;
	- двойное закрытие сессии - неудача;
	- повторная регитсрация - успех;
	- повторное закрытие сессии - успех;
	- проверка регистриации в завершенном сеансе - неудача;

NOTE: проверка авторизации производится обработчиком сообщений, поэтому во всех последующих тестах предполагается, что пользователь зарегистрирван в VHSM 
	
DIGEST SHA1

Вычисляется хеш случайного сообщения размера 4 Кб. Тесты:
	- регистрация пользователя - успех;
	- вызов digestInit - успех;
	- вызов digestInit для неподдерживаемого метода - неудача;
	- вызов digestUpdate - успех;
	- вызов digestUpdate для несуществующей сессии - неудача;
	- вызов digestGetSize - успех;
	- проверка значения, возвращенного digestGetSize - успех;
	- вызов digestGetSize для несуществующей сессии - неудача;
	- вызов digestFinal - успех;
	- повторный вызов digestFinal - неудача;
	- проверка хеша - успех;
	- закрытие сессии - успех;

MAC

Вычисляется HMAC-SHA1 случайного сообщения размера 4 Кб с пустым ключом. Тесты:
	- регистрация пользователя - успех;
	- вызов macInit - успех;
	- вызов macInit для неподдерживаемого метода - неудача;
	- вызов macUpdate - успех;
	- вызов macUpdate для несуществующей сессии - неудача;
	- вызов macGetSize - успех;
	- проверка значения, возвращенного macGetSize - успех;
	- вызов macGetSize для несуществующей сессии - неудача;
	- вызов macFinal - успех;
	- повторный вызов macFinal - неудача;
	- проверка подписи - успех;
	- закрытие сессии - успех;
	
KEY MANAGEMENT

Функции VHSM только делегируют вызовы хранилищу. Тесты выполняют проверку авторизации (для открытой и несуществующей сессии) для всех функций, связанных с управлением ключами:
	- importKey;
	- deleteKey;
	- getKeyIdsCount;
	- getKeyIds;
	- getKeyInfo;
